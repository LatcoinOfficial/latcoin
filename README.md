# Latcoin (LATC)

Latcoin (LATC) es un token de utilidad diseñado para impulsar la adopción de blockchain en América Latina, integrando educación Web3, comercio digital P2P y herramientas de economía descentralizada dentro del ecosistema LatBroker.

LATC nace como un activo utilitario que conecta personas, comercios y comunidades digitales, promoviendo el uso responsable y práctico de la tecnología blockchain.
